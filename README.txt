This is the MATLAB code for the following paper: 

Fast and Robust Multi-View Multi-Task Learning via Group Sparsity. 
A submission to the 28th International Joint Conference on Artificial Intelligence. 



Please run 'demo.m' to demonstrate the weight matrix decomposition by AGILE on a synthetic dataset.

Please run 'demo_exp.m' to show how AGILE performs on a dual-heterogeneous dataset.

For more information on AGILE, please type 'help AGILE_train' or 'help AGILE_test' under MATLAB prompt.
